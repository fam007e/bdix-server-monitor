## üêõ Reporting Bugs

If you find a bug, please help us by reporting it on our [GitHub Issues](https://github.com/fam007e/bdix-server-monitor/issues) page.

**Before submitting a bug report, please:**
1.  **Search existing issues**: Check if the bug has already been reported.
2.  **Gather information**: Note down steps to reproduce, expected vs. actual behavior, error messages, your operating system, compiler version, and `bdix-monitor` version (`./bin/bdix-monitor -V`).
3.  **Use the bug report template**: When creating a new issue, select the "Bug Report" template to provide all necessary details.
